How to install animBot:

Just drag the file "animBot Drag'n Drop Install.mel" into Maya's viewport.

Alternatively, you can copy the animBot folder into your scripts folder:
(Linux)$HOME/maya/scripts
(Mac OS X) $HOME/Library/Preferences/Autodesk/maya/scripts
(Windows) \Users\<username>\Documents\maya\scripts
And run this python code:
import animBot; animBot.welcome()


More information at help.animbot.ca
If you are installing Enterprise Edition, please do not follow these instructions.
